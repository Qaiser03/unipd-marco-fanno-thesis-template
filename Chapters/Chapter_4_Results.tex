\section{Illustrative Code Listing}

\cref{lst:sample-code} shows the \texttt{minted} package rendering a short script with syntax highlighting. \textbf{Note:} \texttt{minted} requires Python + Pygments installed and \texttt{-shell-escape} enabled at compile time -- see README.md if this block fails to compile.

\begin{figure}[H]
\begin{minted}{python}
import numpy as np

def simulate_gbm(p0, mu, sigma, steps, dt, seed=0):
    rng = np.random.default_rng(seed)
    path = np.empty(steps)
    path[0] = p0
    for t in range(1, steps):
        shock = rng.normal(0, np.sqrt(dt))
        path[t] = path[t - 1] * (1 + mu * dt + sigma * shock)
    return path
\end{minted}
\caption{Illustrative code listing (minted)}
\label{lst:sample-code}
\end{figure}
